Lambda Functions in Python

What are Lambda Functions?

Lambda functions are small, anonymous functions that can have any number of arguments, but only one expression.

Basic Syntax

# Regular function def square(x): return x * x # Lambda function square = lambda x: x * x print(square(5)) # Output: 25

Motivation Behind

One-liner: Single expression only
Anonymous: No function name needed
Inline: Can be used directly in code
Functional: Perfect for sorted, filter

🔤 Variable Assignment

f stores the lambda function object, just like any other variable.

f = lambda a: a * a print(f(5)) # 25

⚡ Lambda Keyword

lambda tells Python this is an anonymous function.

# Same as: def f(a): return a * a f = lambda a: a * a

📥 Parameter

a is the parameter that receives the input value.

f(5) # a = 5 f(10) # a = 10

🎯 Expression

a * a is the single expression that gets evaluated and returned.

# Only one expression allowed! lambda a: a * a # ✅ lambda a: a * a; print(a) # ❌

Common Use Cases

Lambda functions with built-in functions

With filter()

numbers = [1, 2, 3, 4, 5, 6] evens = list(filter(lambda x: x % 2 == 0, numbers)) print(evens) # [2, 4, 6]

With sorted()

students = [("Alice", 85), ("Bob", 92)] sorted_by_grade = sorted(students, key=lambda x: x[1]) print(sorted_by_grade)

Multiple Arguments

add = lambda x, y: x + y result = add(3, 5) print(result) # 8
