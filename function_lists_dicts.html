Receiving Arguments

Lists and Dictionaries as Parameters

Lists as Arguments

Pass entire lists to functions for processing multiple values.

List Processing

def process_numbers(numbers): total = sum(numbers) average = total / len(numbers) return total, average scores = [85, 92, 78, 96, 88] total, avg = process_numbers(scores) print(f"Total: {total}, Average: {avg}")

Dictionaries as Arguments

Pass dictionaries to functions for structured data processing.

Dictionary Processing

def process_student(student): name = student["name"] grade = student["grade"] return f"{name} got {grade}" student = {"name": "Alice", "grade": "A"} result = process_student(student) print(result)

Advanced Examples

Real-world data processing scenarios

List of Dictionaries

def analyze_students(students): total_grades = 0 for student in students: total_grades += student["grade"] return total_grades / len(students) students = [ {"name": "Alice", "grade": 85}, {"name": "Bob", "grade": 92} ] avg = analyze_students(students)

Dictionary with Lists

def process_data(data): scores = data["scores"] name = data["name"] return f"{name}: {max(scores)}" data = { "name": "Alice", "scores": [85, 92, 78] } result = process_data(data)
