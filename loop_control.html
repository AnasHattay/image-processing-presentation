Loop Control: break, continue & else

Break Statement

Early Exit from Loop
for i in range(10): if i == 5: break # Exit loop immediately print(i)
Output: 0, 1, 2, 3, 4
Finding First Match
numbers = [2, 4, 7, 8, 10, 15] target = 7 for num in numbers: if num == target: print(f"Found {target}!") break print(f"Checking {num}") print("Search complete")
Output: Checking 2, Checking 4, Found 7!, Search complete

Continue Statement

Skip Current Iteration
for i in range(10): if i % 2 == 0: # Skip even numbers continue print(f"Odd number: {i}")
Output: Odd number: 1, 3, 5, 7, 9

Else Clause in Loops

For Loop with Else
for i in range(5): print(f"Processing {i}") else: print("Loop completed normally!") # else runs only if loop wasn't broken
Output: Processing 0, 1, 2, 3, 4, Loop completed normally!
Search with Else
numbers = [1, 3, 5, 7, 9] target = 6 for num in numbers: if num == target: print(f"Found {target}!") break else: print(f"{target} not found in list")
Output: 6 not found in list
While Loop with Else
count = 0 while count < 3: print(f"Count: {count}") count += 1 else: print("While loop finished normally")
Output: Count: 0, 1, 2, While loop finished normally

Key Concepts

break

Exits the loop immediately, skipping any remaining iterations and the else clause

continue

Skips the rest of the current iteration and moves to the next iteration

else

Executes only if the loop completes normally (not broken)
