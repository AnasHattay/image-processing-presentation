While Loops: Conditional Iteration

Basic While Loop Syntax

Simple Counter
count = 0 while count < 5: print(f"Count: {count}") count += 1 # Don't forget to update!
Output: Count: 0, 1, 2, 3, 4
User Input Loop
user_input = "" while user_input.lower() != "quit": user_input = input("Enter 'quit' to exit: ") if user_input.lower() != "quit": print(f"You entered: {user_input}")
Continues until user types 'quit'

While Loop Best Practices

Avoid Infinite Loops

Always ensure your condition will eventually become False

count = 0 while count < 10: print(count) # count += 1 # MISSING! Infinite loop!
Use For When Possible

If you know the number of iterations, use a for loop instead

# Better as for loop for i in range(10): print(i)
While vs For

While: When you don't know how many iterations
For: When iterating over sequences
