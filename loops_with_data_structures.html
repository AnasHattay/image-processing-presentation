Loops with Data Structures

Lists - Ordered Collections
Basic List Iteration
fruits = ['apple', 'banana', 'cherry'] # Iterate over values for fruit in fruits: print(fruit) # Iterate with index for i, fruit in enumerate(fruits): print(f"{i}: {fruit}") # Iterate with range (when you need index) for i in range(len(fruits)): print(f"{i}: {fruits[i]}")
Output: 0: apple, 1: banana, 2: cherry
Dictionaries - Key-Value Pairs
Dictionary Iteration Methods
student_grades = { 'Alice': 95, 'Bob': 87, 'Charlie': 92 } # Iterate over keys (default) for name in student_grades: print(name) # Iterate over values for grade in student_grades.values(): print(grade) # Iterate over key-value pairs for name, grade in student_grades.items(): print(f"{name}: {grade}") # Iterate with enumeration for i, (name, grade) in enumerate(student_grades.items()): print(f"{i+1}. {name}: {grade}")
Output: 1. Alice: 95, 2. Bob: 87, 3. Charlie: 92
Sets - Unique Collections
Set Operations in Loops
colors = {'red', 'green', 'blue', 'yellow'} warm_colors = {'red', 'orange', 'yellow'} # Basic iteration for color in colors: print(color) # Check membership during iteration for color in colors: if color in warm_colors: print(f"{color} is a warm color") else: print(f"{color} is a cool color") # Set operations common = colors & warm_colors # Intersection for color in common: print(f"Common color: {color}")
Common colors: red, yellow
Multiple Data Structures
# Combining different data structures students = ['Alice', 'Bob', 'Charlie'] grades = [95, 87, 92] subjects = ('Math', 'Science', 'English') # Using zip to iterate multiple sequences for student, grade, subject in zip(students, grades, subjects): print(f"{student}: {grade} in {subject}") # Nested data structures class_data = { 'students': ['Alice', 'Bob'], 'grades': [95, 87], 'subjects': ['Math', 'Science'] } for key, values in class_data.items(): print(f"{key}:") for value in values: print(f" {value}")
Alice: 95 in Math, Bob: 87 in Science
