List Comprehensions: Pythonic Loops

Traditional Loop vs Comprehension
# Traditional way squares = [] for x in range(5): squares.append(x**2) # List comprehension squares = [x**2 for x in range(5)]
Both produce: [0, 1, 4, 9, 16]

List Comprehension Patterns

Basic Syntax
[expression for item in iterable] # Examples: numbers = [1, 2, 3, 4, 5] doubled = [x * 2 for x in numbers] words = ['hello', 'world', 'python'] upper = [word.upper() for word in words]
doubled: [2, 4, 6, 8, 10]
upper: ['HELLO', 'WORLD', 'PYTHON']
With Conditions (Filtering)
# Only even numbers evens = [x for x in range(10) if x % 2 == 0] # Only positive numbers numbers = [-2, -1, 0, 1, 2, 3] positive = [x for x in numbers if x > 0] # Words longer than 3 characters words = ['cat', 'dog', 'elephant', 'ant'] long_words = [w for w in words if len(w) > 3]
evens: [0, 2, 4, 6, 8]
positive: [1, 2, 3]
long_words: ['elephant']
