Lists: The BEST Container

Basic Examples

List containing different data structures

# A list containing everything! mixed_list = [ 42, # int "hello", # string [1, 2, 3], # list (4, 5, 6), # tuple {7, 8, 9}, # set {'key': 'value'}, # dictionary True, # boolean 3.14 # float ] # Check the list properties print(f"Length: {len(mixed_list)}") # Get the type of each item types_list = [type(item).__name__ for item in mixed_list] print(f"Types: {types_list}")

Output:

Length: 8 Types: ['int', 'str', 'list', 'tuple', 'set', 'dict', 'bool', 'float']

Nested Lists (Lists of Lists)

# 2D matrix using nested lists matrix = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # Access element: row 1, column 2 print(matrix[1][2]) # Modify nested list matrix[0].append(10) print(matrix[0])

Output:

6 [1, 2, 3, 10]

Memory Objects

@0x1001

"hello" (str)
@0x1003
[1, 2, 3] (list)
@0x1004
{'a': 1} (dict)

The list doesn't store the actual objects, only references to their memory locations.

Important: When lists contain mutable objects (other lists, sets, dicts), changes to those objects will be reflected everywhere they're referenced.
